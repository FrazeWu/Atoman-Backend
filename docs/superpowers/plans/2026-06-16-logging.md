# Logging Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add terminal-preserving file logging split into access, app, and error log files.

**Architecture:** Add a focused logging setup unit under `cmd/start_server` so the main server initializes outputs before any startup log line. Standard library app logs write to stdout plus `app.log`, Gin access logs write to stdout plus `access.log`, and fatal/server error logs write to stderr plus `error.log`. `LOG_DIR` controls the directory and defaults to `./log`.

**Tech Stack:** Go standard library `log`, `io`, `os`, Gin `DefaultWriter` / `DefaultErrorWriter`, existing `go test` / `go build` commands.

---

## File Structure

- Create `cmd/start_server/logging.go`: owns log directory resolution, file opening, multi-writer setup, and fatal logger wrapper.
- Create `cmd/start_server/logging_test.go`: tests default path, env override, multi-writer behavior, and failure behavior.
- Modify `cmd/start_server/main.go`: initialize logging at process start, defer close, replace direct fatal calls with the fatal logger, and replace `gin.Default()` with explicit logger/recovery middleware so writers are configured before use.
- Modify `.env.example`: document `LOG_DIR=./log`.
- Optionally modify production Compose outside this repo if the production backend service file exists in the parent deployment repo: set `LOG_DIR=/app/logs` and map `./log:/app/logs`.

## Scope Notes

This plan implements the approved “按调用入口” split. Existing `WARN:` text remains in `app.log`; this plan does not introduce warning-level parsing or log rotation.

	}

		t.Fatalf("expected %s to contain %q, got %q", path, want, string(content))

Create `cmd/start_server/logging.go`:

```go
package main

import (
	"errors"
	"fmt"
	"io"
	"log"
	"os"
	"path/filepath"

	"github.com/gin-gonic/gin"
)

var errCreateLogDir = errors.New("create log directory")

type loggingConfig struct {
	Dir    string
	Stdout io.Writer
	Stderr io.Writer
}

type serverLogs struct {
	AppFile     *os.File
	AccessFile  *os.File
	ErrorFile   *os.File
	FatalLogger *log.Logger
}

func resolveLogDir() string {
	if dir := os.Getenv("LOG_DIR"); dir != "" {
		return dir
	}
	return "./log"
}

func setupLogging(cfg loggingConfig) (*serverLogs, error) {
	if cfg.Dir == "" {
		cfg.Dir = resolveLogDir()
	}
	if cfg.Stdout == nil {
		cfg.Stdout = os.Stdout
	}
	if cfg.Stderr == nil {
		cfg.Stderr = os.Stderr
	}

	if err := os.MkdirAll(cfg.Dir, 0o755); err != nil {
		return nil, fmt.Errorf("%w %s: %v", errCreateLogDir, cfg.Dir, err)
	}

	logs := &serverLogs{}
	var err error
	logs.AppFile, err = openLogFile(cfg.Dir, "app.log")
	if err != nil {
		return nil, err
	}
	logs.AccessFile, err = openLogFile(cfg.Dir, "access.log")
	if err != nil {
		_ = logs.Close()
		return nil, err
	}
	logs.ErrorFile, err = openLogFile(cfg.Dir, "error.log")
	if err != nil {
		_ = logs.Close()
		return nil, err
	}

	log.SetOutput(io.MultiWriter(cfg.Stdout, logs.AppFile))
	gin.DefaultWriter = io.MultiWriter(cfg.Stdout, logs.AccessFile)
	gin.DefaultErrorWriter = io.MultiWriter(cfg.Stderr, logs.ErrorFile)
	logs.FatalLogger = log.New(io.MultiWriter(cfg.Stderr, logs.ErrorFile), "", log.Flags())

	return logs, nil
}

func openLogFile(dir string, name string) (*os.File, error) {
	path := filepath.Join(dir, name)
	file, err := os.OpenFile(path, os.O_CREATE|os.O_WRONLY|os.O_APPEND, 0o644)
	if err != nil {
		return nil, fmt.Errorf("open log file %s: %w", path, err)
	}
	return file, nil
}

func (l *serverLogs) Close() error {
	if l == nil {
		return nil
	}

	var errs []error
	if l.AppFile != nil {
		errs = append(errs, l.AppFile.Close())
	}
	if l.AccessFile != nil {
		errs = append(errs, l.AccessFile.Close())
	}
	if l.ErrorFile != nil {
		errs = append(errs, l.ErrorFile.Close())
	}
	return errors.Join(errs...)
}
```

with:

```go
func main() {
	logs, err := setupLogging(loggingConfig{})
	if err != nil {
		log.Fatalf("Failed to initialize logging: %v", err)
	}
	defer func() {
		if err := logs.Close(); err != nil {
			log.Printf("WARN: failed to close log files: %v", err)
		}
	}()
	fatalLogger := logs.FatalLogger

	log.Println("Starting Atoman Backend Server...")
```
